## Model Architecture

```Python
# Model Architecture

Sequential(
    (0): Linear(in_features=85, out_features=256)
    (1): ReLU(inplace)
    (2): Linear(in_features=256, out_features=128)
    (3): ReLU(inplace)
    (4): Linear(in_features=128, out_features=64)
    (5): ReLU(inplace)
    (6): Linear(in_features=64, out_features=3)
)
```
